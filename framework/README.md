# Framework

テンプレートとスクリプトを管理するディレクトリです。

## templates/

研究コンテンツのテンプレートを配置します。

| ファイル        | 用途                         |
| --------------- | ---------------------------- |
| `memo.md`       | 日次メモのテンプレート       |
| `theme.md`      | 研究テーマ定義のテンプレート |
| `paper-note.md` | 論文ノートのテンプレート     |

### カスタマイズ

テンプレートを編集して、自分のワークフローに合わせてください。
フロントマター（YAML部分）のフィールドを追加・削除できます。

## scripts/

ユーティリティスクリプトを配置します。

### new-daily.sh

日次メモディレクトリを作成します。

```bash
./new-daily.sh "タイトル"
```

**出力**:

```
content/daily/YYYY-MM-DD_タイトル/
├── memo.md           # テンプレートから生成
└── generated/        # AI生成コンテンツ用
```
