AIエージェントと協働して研究を進めるためのワークスペースです。

## 特徴

- **並列エージェント**: 複数AIが同時に論文調査
- **6つの専門スキル**: 調査、整理、提案を自動化
- **ナレッジ管理**: themes/papers/dailyの3層構造
