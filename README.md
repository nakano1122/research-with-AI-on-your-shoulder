## なぜこのプロジェクト？

研究者の時間は貴重です。論文調査、アイデア整理、進捗管理...これらをAIエージェントが並列で処理します。

<p align="center">
  <img src="docs/images/parallel-agents.png" alt="Parallel Agents">
</p>

## 6つの専門スキル

| スキル | できること |
|--------|------------|
| `/research-partner` | 研究の相談相手。今日やることを一緒に整理 |
| `/survey-broad` | 並列エージェントで広範囲の論文を一気に調査 |
| `/survey-focused` | あなたのテーマに絞った深い調査 |
| `/idea-organizer` | 散らばったアイデアを構造化・図解 |
| `/related-work` | メモから関連研究を自動で発見 |
| `/next-action` | 進捗を分析し、次の一手を提案 |

## 始め方

```bash
pip install pre-commit && pre-commit install
```

Claude Codeを開いて `/research-partner` と入力するだけ。
