# Themes

研究テーマの定義を配置するディレクトリです。

## AIがどう参照するか

AIエージェントは、スキル実行時にこのディレクトリを読み込み、あなたの研究分野を理解します。

- `/survey-focused`: テーマに沿った論文調査
- `/related-work`: テーマに関連する研究の調査
- `/next-action`: 研究目標に基づいたアクション提案

## テーマの書き方

`framework/templates/theme.md` を参考に、以下の項目を記述してください:

- **概要**: このテーマの目的と背景
- **研究課題**: 解決したい課題のリスト
- **キーワード**: 検索に使用するキーワード
- **関連分野**: 関係する研究分野
- **参考文献**: 重要な参考文献

## 複数テーマの管理

複数の研究テーマがある場合は、それぞれ別ファイルとして作成してください。
AIは全てのテーマファイルを読み込んで、文脈を理解します。
