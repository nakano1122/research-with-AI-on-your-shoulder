# Next Action

進捗を分析し、次にすべきアクションを提案する。

## ワークフロー

```
1. 状況把握
   → content/themes/ で目標を確認
   → content/daily/ で最近の活動を確認

2. ギャップ分析
   → 目標と現状の差分を特定
   → 未完了タスクを抽出

3. アクション提案
   → 優先度付きの具体的アクションを提示
   → ロードマップを図解化

4. コミットメント
   → ユーザーと次のステップを合意
```

## 状況把握

```bash
# 目標を確認
Read content/themes/_index.md
Glob content/themes/*.md

# 最近の活動を確認
Glob content/daily/*/memo.md | sort -r | head -5
Grep "TODO|次|予定" content/daily/
```

## 未完了タスク抽出

メモから未完了タスクを抽出:

```bash
# チェックボックス形式のTODOを検索
Grep "- \\[ \\]" content/daily/
```

## アクション提案フレームワーク

| 優先度 | アクション | 理由           | 所要時間 |
| ------ | ---------- | -------------- | -------- |
| 高     | ...        | 目標達成に直結 | 1h       |
| 中     | ...        | 進捗に貢献     | 2h       |
| 低     | ...        | あれば良い     | 30m      |

## ロードマップ可視化

`/drawio` スキルでロードマップを作成:

```
/drawio ガントチャート風ロードマップを作成
- 今週のタスク
- 来週の予定
- マイルストーン
```

出力先: `content/daily/{今日}/generated/roadmap.drawio`

## 出力形式

```markdown
# 次のアクション: {日付}

## 現状サマリー

- 目標: ...
- 最近の進捗: ...
- 課題: ...

## 推奨アクション（優先度順）

### 今日やるべきこと

1. **{アクション}** (所要時間: Xh)
   - 理由: ...
   - 完了条件: ...

### 今週中にやること

1. ...

## ロードマップ

![ロードマップ](generated/roadmap.drawio)

## 一言アドバイス

{励ましや具体的なコツ}
```

## 対話のコツ

- 「今一番気になっていることは？」から始める
- 大きなタスクは小さく分解して提示
- 達成可能な範囲で提案（過負荷を避ける）
