# 日次メモインデックス

このディレクトリには、日々の研究メモを配置します。
各メモは `YYYY-MM-DD_{タイトル}/` ディレクトリとして管理され、AIが生成したコンテンツも同じディレクトリに保存されます。

## メモの追加方法

`framework/scripts/new-daily.sh {タイトル}` を実行すると、新しいメモディレクトリが作成されます。

## ディレクトリ構造

```
content/daily/
└── YYYY-MM-DD_{タイトル}/
    ├── memo.md           # あなたのメモ
    └── generated/        # AIが生成したコンテンツ
        └── *.drawio      # 図解など
```
