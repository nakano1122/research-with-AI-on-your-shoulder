# Papers

論文まとめを配置するディレクトリです。

## 論文ノートの書き方

`framework/templates/paper-note.md` をコピーして、以下の項目を記述してください:

- **一言まとめ**: この論文の貢献を1文で
- **研究課題**: この論文が解こうとしている問題
- **手法**: 提案手法の概要
- **結果**: 主要な結果
- **自分の研究との関連**: 自分の研究にどう活かせるか
- **疑問・コメント**: 読んで生じた疑問や批判的コメント

## ファイル命名規則

`{著者名}-{年}-{キーワード}.md`

例:

- `vaswani-2017-attention.md`
- `devlin-2019-bert.md`

## 読書状況の管理

フロントマターの `read_status` フィールドで管理:

- `unread`: 未読
- `reading`: 読書中
- `completed`: 読了

`_index.md` に読書状況別のリストを記載しています。
