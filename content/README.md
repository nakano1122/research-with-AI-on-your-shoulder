# Content

研究コンテンツを管理するディレクトリです。
ユーザーとAIエージェントが日常的に読み書きします。

## ディレクトリ構成

| ディレクトリ | 内容 |
|-------------|------|
| `themes/` | 研究テーマの定義。AIがあなたの研究分野を理解するために参照します。 |
| `papers/` | 読んだ論文のまとめ。文献管理に使用します。 |
| `daily/` | 日次メモ。AIが生成したコンテンツも同じディレクトリに保存されます。 |

## ファイル命名規則

### themes/
- `{テーマ名}.md`

### papers/
- `{著者名}-{年}-{キーワード}.md`

### daily/
- `YYYY-MM-DD_{タイトル}/memo.md` - ユーザーメモ
- `YYYY-MM-DD_{タイトル}/generated/` - AI生成コンテンツ
