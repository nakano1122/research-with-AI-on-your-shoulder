# Survey Focused

ユーザーの研究テーマに絞った深い論文調査を行う。

## ワークフロー

```
1. テーマ確認
   → content/themes/*.md を読み込み
   → ユーザーに調査対象を確認

2. 深掘り検索
   → テーマ固有のキーワードで詳細検索
   → 引用ネットワークを辿る

3. 既存文献との関連付け
   → content/papers/ の既存ノートと比較

4. 整理・出力
   → 論文ノートのドラフト作成
```

## テーマ参照

```bash
# テーマファイルを読み込み
Read content/themes/_index.md
Glob content/themes/*.md

# テーマから抽出する情報
- keywords: 検索キーワード
- 研究課題: 解決したい問題
- 関連分野: 隣接領域
```

## 検索戦略

1. **コアキーワード検索**: テーマの主要キーワードで検索
2. **引用追跡**: 重要論文の引用・被引用を辿る
3. **著者追跡**: キーパーソンの他の論文を確認

## 論文ノート作成

発見した重要論文は `content/papers/` にノートを作成:

```bash
# テンプレートを使用
cp framework/templates/paper-note.md content/papers/{著者}-{年}-{キーワード}.md
```

## 出力形式

調査結果をユーザーに報告:

```markdown
## 調査結果: {テーマ名}

### 最も関連性の高い論文

1. **[タイトル](URL)** - {なぜ関連があるか}

### あなたの研究との接点

- 類似点: ...
- 差別化できる点: ...

### 推奨アクション

- [ ] 詳細に読むべき論文
- [ ] 追加で調べるべきキーワード
```
